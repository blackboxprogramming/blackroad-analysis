## Setup

### 1. Canva Access Token
1. Go to: https://www.canva.dev/
2. Create app and get access token
3. Add as GitHub secret: `CANVA_ACCESS_TOKEN`

### 2. Deploy Enhanced Bot
```bash
# Copy enhanced workflow to all repos
for repo in $(gh repo list BlackRoad-OS --limit 100 --json name --jq '.[].name'); do
  gh repo clone BlackRoad-OS/$repo /tmp/$repo
  mkdir -p /tmp/$repo/.github/workflows
  cp ~/blackroad-bot-workflows/bot-docs-update-with-canva.yml \
     /tmp/$repo/.github/workflows/
  cd /tmp/$repo
  git checkout -b bot/add-visual-docs
  git add .github/workflows/bot-docs-update-with-canva.yml
  git commit -m "Add visual documentation bot with Canva"
  git push origin bot/add-visual-docs
  gh pr create --title "Add visual documentation automation" --body "Adds Canva-powered visual docs"
  cd -
  rm -rf /tmp/$repo
done
```

### 3. Test
Push code changes and watch the bot create visual docs!
